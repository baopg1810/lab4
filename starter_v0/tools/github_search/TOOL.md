# github_search

Tìm kiếm các kho lưu trữ mã nguồn mở trên GitHub.

## Notes
- Gọi API của GitHub `/search/repositories`.
- Trả về danh sách repository kèm theo các thông tin: full_name (tiêu đề), html_url (đường dẫn), số sao (stars), số lượng forks, và mô tả ngắn.
